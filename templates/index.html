Название

eprintf - выводит сообщение об ошибке в stderr

Краткий обзор 

#include <cs50.h> 
int eprintf(string format, ...);

Описание

Со стандартной ошибкой еще выводит сообщение об ошибке, отформатированное как printf(3), добавляя к нему префикс, содержащий номер файла и строки, из которого была вызвана функция. Подробнее о форматировании см. в printf(3)

Возвращаемое значение

Возвращает int, эквивалентный строке, считанной из stdin в [INT_MIN, INT_MAX). Если строка не может быть прочитана, возвращается INT_MAX

Примеры

int main(void)
{
    eprintf("This line of code should be executed\n");

    int i;
    for (i = 0; i < 50; i++)
    {
        printf("i is %d\n", i);
    }

    eprintf("Expected i to be 49 but it is actually %d\n", i);
}
